As estruturas de repetição, também conhecidas como loops, permitem executar um bloco de código várias vezes. Elas são essenciais para automatizar tarefas que se repetem, tornando seus programas mais eficientes.

O Conceito: `for` e `while`

Em Python, as principais estruturas de repetição são for e while.

  • for (para): Usado para iterar sobre uma sequência (como listas, tuplas, dicionários, strings) ou outros objetos iteráveis. Ideal quando você sabe o número de vezes que a repetição deve ocorrer.
  • while (enquanto): Repete um bloco de código enquanto uma condição específica for verdadeira. Ideal quando você não sabe o número exato de repetições e a execução depende de uma condição.

Aplicação e Exemplo de Código com `for`

O loop for é frequentemente usado para percorrer coleções de itens. Veja um exemplo que imprime cada número de 1 a 5:

Exemplo: Loop `for` com `range()`

# Exemplo de Estrutura de Repetição com 'for'

print("Contando de 1 a 5:")
for numero in range(1, 6): # range(1, 6) gera números de 1 a 5
    print(numero)

print("\nIterando sobre uma lista:")
frutas = ["maçã", "banana", "cereja"]
for fruta in frutas:
    print(f"Eu gosto de {fruta}.")

Este código usa for para contar até 5 e para exibir cada fruta em uma lista. O range(1, 6) gera uma sequência de 1 a 5.

Aplicação e Exemplo de Código com `while`

O loop while continua executando enquanto uma condição for verdadeira. É importante ter uma forma de a condição se tornar falsa para evitar loops infinitos.

Exemplo: Loop 'while' para Contagem Regressiva

# Exemplo de Estrutura de Repetição com 'while'

contador = 5
print("Contagem regressiva:")
while contador > 0:
    print(contador)
    contador -= 1 # Decrementa o contador
print("Lançar!")

# Exemplo de loop 'while' com entrada do usuário
print("\nTentativa de login (digite 'sair' para parar):")
senha_correta = "python123"
tentativas = 0
while tentativas < 3:
    senha_digitada = input("Digite a senha: ")
    if senha_digitada == senha_correta:
        print("Login bem-sucedido!")
        break # Sai do loop se a senha estiver correta
    elif senha_digitada.lower() == 'sair':
        print("Saindo do programa.")
        break
    else:
        print("Senha incorreta. Tente novamente.")
        tentativas += 1
else: # Este 'else' é executado se o loop 'while' terminar normalmente (sem 'break')
    print("Número máximo de tentativas excedido.")

Neste exemplo, o loop while conta regressivamente. No segundo exemplo, ele continua pedindo uma senha até que seja correta, até que o usuário digite 'sair' ou até 3 tentativas falhas.

Instruções de Controle de Loop ('break','continue')

  • break: Termina o loop imediatamente.
  • continue: Pula a iteração atual do loop e continua para a próxima.
Exemplo: 'break' e 'continue'

# Exemplo com 'break'
print("Exemplo com break:")
for i in range(1, 10):
    if i == 5:
        break # O loop para quando i for 5
    print(i) # Irá imprimir 1, 2, 3, 4

# Exemplo com 'continue'
print("\nExemplo com continue (pula o ímpar):")
for i in range(1, 6):
    if i % 2 != 0: # Se for ímpar
        continue # Pula para a próxima iteração
    print(i) # Irá imprimir 2, 4
