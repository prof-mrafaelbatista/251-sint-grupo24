Em Python, o conceito de vetores e matrizes é geralmente implementado usando listas. Uma lista é uma coleção ordenada e mutável de itens. Para representar matrizes, usamos listas de listas (listas aninhadas).

O Conceito: Listas em Python

Uma lista é uma estrutura de dados fundamental em Python que pode armazenar uma sequência de itens de diferentes tipos. Ela é definida usando colchetes [] e os itens são separados por vírgulas.

Características importantes das listas:

  • Ordenadas: Os itens têm uma ordem definida e essa ordem não muda.
  • Mutáveis: Você pode alterar, adicionar ou remover itens após a criação da lista.
  • Permitem Duplicatas: Uma lista pode ter itens com o mesmo valor.
  • Indexadas: Cada item em uma lista tem um índice (posição), começando do 0.

Aplicação e Exemplo de Código (Listas/Vetores)

Vamos ver como criar e manipular listas em Python:

Exemplo: Operações Básicas com Listas (Vetores)

# Exemplo de Lista (Vetor) em Python

# Criando uma lista
frutas = ["maçã", "banana", "cereja", "laranja"]
print(f"Lista original de frutas: {frutas}")

# Acessando elementos (índices começam do 0)
print(f"Primeira fruta: {frutas[0]}") # Saída: maçã
print(f"Terceira fruta: {frutas[2]}") # Saída: cereja

# Alterando um elemento
frutas[1] = "morango"
print(f"Lista após alterar banana para morango: {frutas}")

# Adicionando elementos
frutas.append("uva") # Adiciona ao final
print(f"Lista após adicionar uva: {frutas}")
frutas.insert(1, "kiwi") # Adiciona na posição 1
print(f"Lista após adicionar kiwi na posição 1: {frutas}")

# Removendo elementos
frutas.remove("maçã") # Remove pelo valor
print(f"Lista após remover maçã: {frutas}")
elemento_removido = frutas.pop(0) # Remove pelo índice (0 é o primeiro) e retorna o elemento
print(f"Lista após remover {elemento_removido} (pelo índice): {frutas}")

# Tamanho da lista
print(f"Número de frutas na lista: {len(frutas)}")

# Iterando sobre a lista (com loop for)
print("\nMinhas frutas favoritas:")
for fruta in frutas:
    print(f"- {fruta}")

Este código demonstra as operações mais comuns com listas, como acesso, alteração, adição e remoção de elementos, além de como percorrer a lista.

O Conceito: Listas Aninhadas (Matrizes) em Python

Uma matriz (ou array 2D) pode ser representada em Python como uma lista onde cada elemento é, por sua vez, outra lista. Isso cria uma estrutura de linhas e colunas.

Exemplo de matriz 2x3 (2 linhas, 3 colunas):


matriz = [
    [1, 2, 3],  # Linha 0
    [4, 5, 6]   # Linha 1
]

Aplicação e Exemplo de Código (Matrizes)

Vamos ver como criar e manipular listas aninhadas:

Exemplo: Operações Básicas com Listas Aninhadas (Matrizes)

# Exemplo de Matriz (Lista Aninhada) em Python

# Criando uma matriz 3x3
matriz = [
    [10, 20, 30],
    [40, 50, 60],
    [70, 80, 90]
]
print("Matriz original:")
for linha in matriz:
    print(linha)

# Acessando elementos: matriz[linha][coluna]
print(f"\nElemento na linha 0, coluna 1: {matriz[0][1]}") # Saída: 20
print(f"Elemento na linha 2, coluna 2: {matriz[2][2]}") # Saída: 90

# Alterando um elemento
matriz[1][1] = 55
print("\nMatriz após alterar o elemento central:")
for linha in matriz:
    print(linha)

# Iterando sobre a matriz (loops aninhados)
print("\nPercorrendo a matriz elemento por elemento:")
for linha in range(len(matriz)):
    for coluna in range(len(matriz[linha])):
        print(f"Elemento na posição ({linha}, {coluna}): {matriz[linha][coluna]}")

Este código mostra como acessar e alterar elementos de uma matriz, e como usar loops aninhados para percorrer todos os seus elementos.
