As estruturas de seleção, ou condicionais, permitem que seu programa tome decisões e execute diferentes blocos de código com base em condições verdadeiras ou falsas. Em Python, usamos if, elif (else if) e else.

O Conceito: 'if', 'elif', 'else'

A ideia principal é "Se (if) esta condição for verdadeira, faça isso. Senão, se (elif) esta outra condição for verdadeira, faça aquilo. Senão (else), faça algo diferente."

  • 'if' (se): O bloco de código sob um 'if' é executado apenas se a condição associada for verdadeira.
  • `elif` (senão se): É usado após um 'if' (ou outro 'elif') para testar uma nova condição se a anterior foi falsa.
  • 'else' (senão): Opcional. O bloco de código sob um 'else' é executado se nenhuma das condições 'if' ou 'elif' anteriores for verdadeira.

Aplicação e Exemplo de Código

Vamos criar um pequeno programa que verifica a idade de uma pessoa e diz se ela é criança, adolescente ou adulto.

Exemplo: Verificação de Idade

#Exemplo de Estrutura de Seleção

idade = 17 # Você pode mudar este valor para testar

if idade < 12:
    print("Você é uma criança.")
elif idade < 18: 
    print("Você é um adolescente.")
else: 
    print("Você é um adulto.")

# Outro exemplo: verificando se um número é par ou ímpar
numero = 7
if numero % 2 == 0:
    print(f"O número {numero} é par.")
else:
    print(f"O número {numero} é ímpar.")

Este código usa if, elif e else para categorizar a idade. No segundo exemplo, verifica se um número é par ou ímpar usando o operador de módulo (%).

Operadores Lógicos ('and', 'or', 'not')

Você pode combinar múltiplas condições usando operadores lógicos:

  • `and` (e): Verdadeiro se TODAS as condições forem verdadeiras.
  • `or` (ou): Verdadeiro se PELO MENOS UMA das condições for verdadeira.
  • `not` (não): Inverte o valor de uma condição (verdadeiro se for falsa, falso se for verdadeira).
Exemplo: Usando 'and' e 'or'

# Exemplo com operadores lógicos

tem_carteira = True
idade = 20

if idade >= 18 and tem_carteira:
    print("Pode dirigir.")
else:
    print("Não pode dirigir.")

temperatura = 28
chovendo = True

if temperatura > 25 or chovendo:
    print("Leve um guarda-chuva ou protetor solar.")
else:
    print("Clima agradável.")
